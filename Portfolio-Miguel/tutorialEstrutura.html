TUTORIAL Sebastian's Tale

Contexto:

Em um belo dia de sol, Sebastian estava caminhando pela cidade de Bellcity e, ao avistar uma biblioteca, resolveu entrar para procurar uma boa leitura. Assim, procurou por livros de ação pois, desde criança, se interessava por leituras empolgantes. Ao passar o olho pelas opções, viu um livro que habitualmente não estava lá... Um frequentador assíduo da biblioteca como Sebastian não deixaria a oportunidade de ler um livro novo de lado. Então, pegando o livro estranho, percebeu que o mesmo estava praticamente todo em branco, porém, ao folhear o livro e voltar para o início, percebeu também que tinha algo escrito que não estava lá antes, dizia: "Precisamos de sua ajuda, por favor!!". Sebastian, ao ler, se deu conta que seu corpo todo estava brilhando como se estivesse sumindo, sendo, assim, puxado para dentro do livro. Ao acordar em um castelo com pessoas desesperadas, Sebastian quis entender o que estava acontecendo e perguntou a quem encontrou à primeira vista, explicaram-o que, para salvar o mundo, eles precisavam abrir uma porta... No entanto, as missões para encontra-lá eram tão difíceis que nínguem do mundo deles jamais conseguiu. Condicionado a voltar ao seu próprio mundo somente se conseguisse concluir as missões propostas, Sebastian resolveu ajudá-los, pois, somente assim, eles teriam energia o suficiente para mandá-lo com os recursos que tem atrás da porta final.














Controles:

W : O jogador movimenta uma unidade para cima;

A : O jogador movimenta uma unidade para esquerda;

S : O jogador movimenta uma unidade para baixo;

D : O jogador movimenta uma unidade para direita;

I : O jogador interage com o objeto que está em cima.

& : Símbolo que representa o jogador.

* : Símbolo que representa uma parede, o jogador ao se movimentar não pode passar pela parede.

@ : Símbolo que representa a chave para abrir a porta para finalizar a fase, a porta abre no momento que o jogador interage com a chave.

D : Símbolo que representa a porta fechada.

= : Símbolo que representa a porta aberta quando o jogador interage com a chave.

O : Símbolo que representa um botão que o usuário pode interagir, o botão fica no chão e o jogador deve ficar em cima dele para poder interagir.

# : Símbolo que representa um espinho. A fase é reiniciada quando o jogador toca no espinho, caso a fase seja reiniciada três vezes, o jogo volta para o menu principal.

> : Símbolo que representa um teletransporte. O teletransporte sempre deve vir em pares, quando o jogador toca em um ele é transportado para o outro e vice-versa.
